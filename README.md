# 🕷️ Async Blog Parser

Асинхронный парсер блога на Python с сохранением данных в SQLite через кастомную ORM.

Проект демонстрирует:
- асинхронный web scraping (aiohttp)
- парсинг HTML (BeautifulSoup)
- простую ORM-архитектуру
- сохранение данных в SQLite
- CLI-интерфейс

---

## ⚙️ Возможности

- 📄 Парсинг статей с сайта
- ⚡ Асинхронная загрузка страниц (aiohttp + asyncio)
- 🔁 Поддержка пагинации
- 🧠 Фильтрация новых статей (без дублей)
- 💾 Сохранение в SQLite через ORM
- 🧩 Простая кастомная ORM система
- 🖥️ CLI меню управления
